# Automated Procurement & Audit System
<img width="2426" height="1432" alt="pern-vendor" src="https://github.com/user-attachments/assets/b2928aca-7818-49db-af62-b0e062276c9e" />

A full-stack skeleton-like financial application designed to manage vendor invoices with a audit trail, Idemopotency key.
<img width="2358" height="1438" alt="pern-admin" src="https://github.com/user-attachments/assets/00c73748-18c3-40be-b387-07843aa8d345" />

Idemopotency key is applied upon invoice creation, Audit log is created if the key succeeds, alongside the approval and denial of Invoices.
<img width="2270" height="1342" alt="pern-audit" src="https://github.com/user-attachments/assets/c2820f66-bd83-4432-8dff-775b810be367" />
Tech stack: React(Vite), Node, Express, PostgreSQL, Docker, Docker Compose

Prereq
Docker and docker compose

Installation:
1.Clone rep
2.docker-compose up --build
